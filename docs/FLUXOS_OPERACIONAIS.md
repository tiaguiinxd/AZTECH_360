## Visão Geral do Ciclo de Vida de um Serviço

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           CICLO DE VIDA DO SERVIÇO                              │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│   PRÉ-SERVIÇO          PREPARAÇÃO           KICK-OFF          EXECUÇÃO         │
│   ───────────          ──────────           ────────          ────────         │
│                                                                                 │
│   [Comercial]    →    [Engenharia]    →    [Todos]     →    [Campo]           │
│   [Técnico]            [RH]                                  [Gestão]          │
│   [Suprimentos]        [Suprimentos]                                           │
│   [Financeiro]                                                                  │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## FASE 1: PRÉ-SERVIÇO (Captação e Proposta)

### 1.1 Captação da Demanda
**Responsável:** Comercial

| Etapa | Ação | Entregável |
|-------|------|------------|
| 1.1.1 | Receber demanda do cliente | Briefing inicial |
| 1.1.2 | Coletar escopo preliminar | Documento de escopo |
| 1.1.3 | Identificar projetos necessários | Lista de projetos |
| 1.1.4 | Registrar oportunidade no sistema | Serviço status "Prospecção" |

### 1.2 Análise Técnica
**Responsável:** Setor Técnico + Suprimentos

| Etapa | Ação | Entregável |
|-------|------|------------|
| 1.2.1 | Analisar viabilidade técnica | Parecer técnico |
| 1.2.2 | Elaborar memorial descritivo | Memorial descritivo |
| 1.2.3 | Definir especificações de materiais | Lista de materiais |
| 1.2.4 | Levantar custos com Suprimentos | Planilha de custos |
| 1.2.5 | Estimar prazo de execução | Cronograma macro |
| 1.2.6 | Elaborar orçamento detalhado | Orçamento técnico |

### 1.3 Elaboração da Proposta
**Responsável:** Comercial + Financeiro

| Etapa | Ação | Entregável |
|-------|------|------------|
| 1.3.1 | Compilar orçamento técnico | Proposta base |
| 1.3.2 | Validar margem com Financeiro | Margem aprovada |
| 1.3.3 | Definir condições comerciais | Termos comerciais |
| 1.3.4 | Elaborar proposta formal | Proposta comercial |

### 1.4 Negociação e Aprovação
**Responsável:** Comercial

| Etapa | Ação | Entregável |
|-------|------|------------|
| 1.4.1 | Apresentar proposta ao cliente | Proposta enviada |
| 1.4.2 | Negociar ajustes se necessário | Proposta revisada |
| 1.4.3 | Obter aprovação do cliente | Pedido/Contrato assinado |
| 1.4.4 | Atualizar status no sistema | Serviço status "Aprovado" |

## FASE 2: PREPARAÇÃO (Planejamento da Execução)

### 2.1 Planejamento de Engenharia
**Responsável:** Engenharia

| Documento | Descrição | Destino |
|-----------|-----------|---------|
| **Cronograma de Execução** | Fases, marcos, datas | Todos os setores |
| **Cronograma de Contratação** | Quando cada função será necessária | RH |
| **Cronograma de Equipamentos** | Mobilização/desmobilização | Suprimentos |
| **Cronograma de Materiais** | Entregas programadas | Suprimentos |
| **Plano de Qualidade** | Inspeções, testes, aceites | Técnico/Campo |
| **Plano de Segurança** | PPRA, APR, DDS | Campo/SESMT |

### 2.2 Requisições para RH
**Responsável:** Engenharia → RH

| Informação | Prazo | Observação |
|------------|-------|------------|
| Quantidade de profissionais por função | Mínimo 30 dias antes | Contratação CLT |
| Período de alocação | - | Início e fim previsto |
| Qualificações necessárias | - | NRs, certificações |
| Local de trabalho | - | Deslocamento, hospedagem |

### 2.3 Requisições para Suprimentos
**Responsável:** Engenharia → Suprimentos

| Informação | Prazo | Observação |
|------------|-------|------------|
| Lista de materiais com datas | Conforme lead time | Considerar prazos de entrega |
| Lista de equipamentos | Mínimo 15 dias antes | Locação ou próprio |
| Ferramentas especiais | - | Calibração, certificação |
| EPIs específicos | - | Por função |

### 2.4 Validação dos Responsáveis
**Responsável:** Gestores de cada setor

| Setor | Validação |
|-------|-----------|
| Engenharia | Cronogramas são factíveis |
| RH | Consegue contratar no prazo |
| Suprimentos | Consegue entregar materiais/equipamentos |
| Financeiro | Fluxo de caixa comporta |

## FASE 3: KICK-OFF (Oficialização)

### 3.1 Reunião de Kick-off
**Participantes:** Todos os envolvidos

**Pauta:**
1. Apresentação do escopo do serviço
2. Cronograma geral
3. Definição da equipe (ver 3.2)
4. Responsabilidades de cada um
5. Canais de comunicação
6. Próximos passos

### 3.2 Definição da Equipe do Serviço
**Registro no Sistema:**

| Função no Serviço | Responsabilidade |
|-------------------|------------------|
| **Comercial (Pós-venda)** | Relacionamento com cliente, medições, aditivos |
| **Suprimentos** | Compras, logística de materiais |
| **Engenheiro Responsável** | Gestão técnica, qualidade, segurança |
| **Planejamento** | Acompanhamento de cronograma, indicadores |
| **Execução (Encarregado)** | Gestão da equipe de campo |
| **Assistente** | Apoio administrativo, documentação |

### 3.3 Liberação para Execução
**Checklist de Liberação:**

- [ ] Contrato/Pedido assinado
- [ ] Equipe definida e comunicada
- [ ] Cronogramas validados
- [ ] Materiais críticos em estoque ou em trânsito
- [ ] Equipe de campo contratada ou alocada
- [ ] Documentação de segurança aprovada
- [ ] Ordem de serviço emitida

**Status no Sistema:** "Em Andamento"
